Engineering Projects

Power Engineering Projects

State Of Charge (SoC) Estimator Of Battery Using Arduino

HARDWARE • ARDUINO

Developed a real-time State-of-Charge (SoC) estimator for Li-ion batteries using the **Coulomb counting method** on **Arduino**. This provides a low-cost, accurate monitoring solution for portable power systems and is validated against physical battery cycling data.

Arduino C++ Current Sensor Portable Power

Hybrid Solar-Wind Energy System with Grid Synchronization Using MATLAB Simulink

SIMULATION • POWER ELECTRONICS

Simulated a robust hybrid energy system combining solar PV and wind generation. Focused on **Grid Synchronization** using advanced PLL control algorithms in **MATLAB Simulink** to ensure stable power injection and reactive power compensation.

MATLAB Simulink PLL PCC Control Renewable Energy

Frequency Glitch Detection And Removal Using DSP Techniques

DSP • ALGORITHM DEVELOPMENT

Developed and implemented **Digital Signal Processing (DSP) techniques** to identify and eliminate transient frequency glitches from noisy sensor data. Used **Kalman Filtering** and adaptive notch filters for precise signal cleanup and reliability improvement.

Kalman Filter Notch Filter C/C++ Embedded DSP

Design Of 11kV Feeders For A New Town

DISTRIBUTION SYSTEM • PLANNING

Engineered the complete design and planning for a new **11kV distribution feeder network** covering a small town. Calculations included load flow analysis, voltage drop assessment, conductor sizing, and substation transformer capacity planning based on demand forecasts.

Load Flow Analysis System Design Feeder Protection ETAP/DigSilent
